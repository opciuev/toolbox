# 🛠️ 实用工具集合

一个功能丰富的在线工具集合网站，提供25+种实用工具，包括随机数生成、文本处理、数据转换、开发者工具等。

## ✨ 功能特色

### 🎲 随机生成 & 统计
- 高级随机数生成器（支持批量、去重、排序）
- 随机颜色生成器（多种格式）
- 安全密码生成器（可定制复杂度）
- 多语言随机名字生成器
- 智能决策器（支持权重）

### 🎮 互动游戏
- 可定制幸运转盘
- 权重抽奖系统
- 多面骰子模拟器
- 猜数字游戏（多难度）

### 💾 数据工具
- UUID生成器（多版本）
- 二维码生成器（可定制样式）
- Base64编解码器
- JSON格式化工具
- 时间戳转换器
- URL编解码工具

### 📝 文本处理
- 智能文本统计分析
- 多种大小写转换
- 高级去重工具
- Lorem Ipsum生成器
- 搜索替换工具
- 文本格式化工具

### 👨‍💻 开发者工具
- 正则表达式测试器
- 高级颜色选择器
- CSS渐变生成器
- 字符编码转换器
- HTML/XML格式化器
- CSS美化工具

## 🚀 技术特色

- **纯前端实现** - 无需服务器，数据本地处理
- **响应式设计** - 完美适配桌面和移动设备
- **现代化UI** - 统一的设计风格和流畅动画
- **隐私保护** - 所有处理都在本地完成
- **零依赖** - 除二维码生成外，无外部依赖

## 📦 部署

### Vercel 部署（推荐）

1. Fork 这个仓库
2. 在 [Vercel](https://vercel.com) 中导入项目
3. 自动部署完成！

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/yourusername/random-tools.git
cd random-tools

# 启动本地服务器
python -m http.server 8000
# 或者使用 Node.js
npx serve .

# 访问 http://localhost:8000
```
